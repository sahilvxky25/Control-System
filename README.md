# Control-System

A collection of MATLAB Live Scripts (`.mlx`) for core Control Systems concepts, analysis methods, and simulator-based experiments.

## Contents

This repository includes topic-wise live scripts such as:

- Introduction and modeling (`Intro.mlx`, `Frequency Domain Modelling.mlx`)
- Time and frequency response (`Time Response.mlx`, `Bode Plot.mlx`, `Nyquist Plot.mlx`)
- Stability and design methods (`Routh-Hurwitz Criterion.mlx`, `Root Locus.mlx`, `Compensator.mlx`, `PID.mlx`)
- System representation (`Signal Flow Graph.mlx`, `Zeros and Poles.mlx`, `Block Reduction.mlx`)
- Practical process/simulator studies (`OVEN.mlx`, `P H Simulator.mlx`, `Pressure Process Control.mlx`, `Digital Control Trainer.mlx`, `Linear System Simulator.mlx`)

## Prerequisites

- MATLAB with support for Live Scripts (`.mlx`)

## How to Use

1. Clone or download this repository.
2. Open any `.mlx` file in MATLAB.
3. Run sections step-by-step to study the concept and results.

## Notes

- Script names are kept as-is from the source material.
- Some filenames may contain spaces or minor spelling variations.
